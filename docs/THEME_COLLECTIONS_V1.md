## Manuell kurering

Följande beslut kräver manuell bedömning och ska INTE automatiseras:

### 1. Hero Theme Selection
- Kräver: Visuell bedömning av "första intryck"
- Frekvens: Vid kollektionsskapande
- Ansvarig: Produktägare/designer

### 2. `tier:premium` Tilldelning
- Kräver: Kvalitetsbedömning utöver varningstal
- Kriterier: Unik identitet, bred appell, teknisk kvalitet
- Frekvens: Vid ny theme-promotion

### 3. `use:poster` / `use:gallery` Tilldelning
- Kräver: Bedömning av visuell impact och konstnärlig kvalitet
- Kan INTE härledas från färgdata
- Frekvens: Vid ny theme-skapande

### 4. Kollektion → Preset Koppling
- Vilka presets ska visa vilka kollektioner?
- Kräver: Produktbeslut baserat på användningsfall
- Exempel: "A2_Paper_v1" → visar "Premium Poster"

### 5. Framtida kollektioner
- Beslut om nya kollektioner (Mood: Cool, Mood: Dark, etc.)
- Baseras på: Användarfeedback, affärsbehov
- Ska INTE skapas automatiskt
