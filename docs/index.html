Demos

Systemet erbjuder två parallella implementationer med olika användningsområden:

Demo A WebGL

Interaktiv kartvy med MapLibre. Snabb design-iteration, stöd för perspektiv och kreativ utforskning.

Öppna webbgränssnitt Exportera A2 (150 DPI)

Demo B Print

Server-side rendering med Mapnik. Produktionskvalitet, kartografisk korrekthet, byte-identiska exports.

Öppna webbgränssnitt Export via API (se nedan)

Vad är Demo A och Demo B?

Demo A - Interaktiv / MapLibre

Demo A använder MapLibre GL JS för att rendera kartan direkt i webbläsaren. Det ger snabb feedback, stöd för 3D-perspektiv (pitch), och är idealiskt för att utforska olika vyer och teman innan final export. Exports sker via Playwright som tar en screenshot av kartan.

Användning: Interaktiv preview, kreativ utforskning, snabb iteration.

Demo B - Print / Mapnik

Demo B använder Mapnik för server-side rendering. Kartan renderas på servern från PostGIS-data, vilket ger exakt kontroll över output. Resultatet är kartografiskt korrekt och deterministiskt - samma input ger alltid exakt samma output (byte-identiskt).

Användning: Final print master, professionella exports, arkivkvalitet.

Vem är systemet för?

Slutanvändare

Vill du skapa vacker kartkonst av Stockholm? Använd webbgränssnittet för att välja tema, område och exportera. Demo A är bäst för att utforska, Demo B för final print.

Print/Export

För professionella exports, använd Demo B's API. Det garanterar reproducerbarhet och kartografisk korrekthet. Format som stöds: PNG, PDF.

Export API

Demo A Export

curl "http://localhost:8082/render?bbox_preset=stockholm_core&theme=paper&render_mode=print&dpi=150&width_mm=420&height_mm=594" --output export.png

Demo B Export

curl -X POST "http://localhost:5000/render" \ -H "Content-Type: application/json" \ -d '{ "bbox_preset": "stockholm_core", "theme": "paper", "render_mode": "print", "dpi": 150, "width_mm": 420, "height_mm": 594, "format": "png" }' --output export.png
