Demos

Systemet erbjuder två parallella implementationer med olika användningsomraden:

Demo A WebGL

Interaktiv kartvy med MapLibre. Snabb design-iteration, stod for perspektiv och kreativ utforskning.

Oppna webbgranssnitt Exportera A2 (150 DPI)

Demo B Print

Server-side rendering med Mapnik. Produktionskvalitet, kartografisk korrekthet, byte-identiska exports.

Oppna webbgranssnitt Export via API (se nedan)

Vad ar Demo A och Demo B?

Demo A - Interaktiv / MapLibre

Demo A anvander MapLibre GL JS for att rendera kartan direkt i webblasaren. Det ger snabb feedback, stod for 3D-perspektiv (pitch), och ar idealiskt for att utforska olika vyer och teman innan final export. Exports sker via Playwright som tar en screenshot av kartan.

Anvandning: Interaktiv preview, kreativ utforskning, snabb iteration.

Demo B - Print / Mapnik

Demo B anvander Mapnik for server-side rendering. Kartan renderas pa servern fran PostGIS-data, vilket ger exakt kontroll over output. Resultatet ar kartografiskt korrekt och deterministiskt - samma input ger alltid exakt samma output (byte-identiskt).

Anvandning: Final print master, professionella exports, arkivkvalitet.

Vem ar systemet for?

Slutanvandare

Vill du skapa vacker kartkonst av Stockholm? Anvand webbgranssnittet for att valja tema, omrade och exportera. Demo A ar bast for att utforska, Demo B for final print.

Print/Export

For professionella exports, anvand Demo B's API. Det garanterar reproducerbarhet och kartografisk korrekthet. Format som stods: PNG, PDF.

Export API

Demo A Export

curl "http://localhost:8082/render?bbox_preset=stockholm_core&theme=paper&render_mode=print&dpi=150&width_mm=420&height_mm=594" --output export.png

Demo B Export

curl -X POST "http://localhost:5000/render" \ -H "Content-Type: application/json" \ -d '{ "bbox_preset": "stockholm_core", "theme": "paper", "render_mode": "print", "dpi": 150, "width_mm": 420, "height_mm": 594, "format": "png" }' --output export.png
